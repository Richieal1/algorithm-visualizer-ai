# 🚀 Setup & Run

## 1. Install Dependencies
```bash
npm install
```

## 2. Start Dev Server
```bash
npm run dev
```

The app will open at `http://localhost:3000`

## 3. For AI Chat Feature
Open the **AI Chat** tab and start asking. The chat now uses Puter.js AI directly from the browser, so no API key is required.

## 4. Features
- **Binary Search** - Visual search algorithm on sorted array
- **Bubble Sort** - See elements bubble to their position
- **Selection Sort** - Find and select minimum elements
- **AI Chat** - Ask questions about algorithms & programming without an API key

## Troubleshooting
- If UI is blank: Check browser console (F12) for errors
- If components don't show: Clear browser cache and hard refresh (Ctrl+Shift+R)
- If AI errors: Check your internet connection and reload the page
